# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please email dylan@aerlixconsulting.com. We take security seriously and will respond promptly.

### Responsible Disclosure

We request that you do not publicly disclose issues until we have had a chance to address them.

## Supported Versions

This project is actively maintained. We will backport critical fixes as necessary.

## Security Controls

This project includes environment-based configuration, secrets isolation, code scanning, and dependency monitoring to minimize risk.
